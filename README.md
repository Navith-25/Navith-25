### 🏆 Featured Projects

#### 🚌 Lanka Transit
*A comprehensive public transit management and ticketing mobile ecosystem.*
* **The Build:** Architected a robust backend using **Java Spring Boot** and developed a cross-platform mobile application using **Flutter (Dart)**.
* **Features:** Integrated live mapping, digital ticketing with QR code scanning, and role-based access control for admins, bus owners, drivers, and passengers.

#### 🎯 TalentScout AI
*An intelligent candidate prioritization system designed to streamline and optimize the recruitment process.*
* **The Architecture:** Developed a scalable backend using **Java Spring Boot** to handle complex data processing and candidate evaluation logic efficiently. 
* **The Impact:** Automated the initial screening phases, allowing HR teams to focus on top-tier candidates through smart data sorting and RESTful API integrations.

#### 💸 [Hybrid Personal Finance Manager](https://github.com/Navith-25/financemanager)
*An offline-first application architected to handle data synchronization between local and cloud databases.*
* **Dual-Database Architecture:** Engineered a Spring Boot backend connecting to **SQLite** (for local, offline storage) and **Oracle Database** (for central analytics) simultaneously.
* **Custom Sync Engine:** Wrote a robust `SyncService` that detects unsynced local records and pushes them to the cloud using **Oracle Stored Procedures**.

#### 📡 Smart IoT Mailbox
*A hardware-software integration project bringing traditional mail into the digital age.*
* **The Build:** Utilized **ESP32** microcontrollers and physical sensors to detect mail delivery in real-time and transmit actionable alerts.
